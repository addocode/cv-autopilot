# Contributing

Dieses Repository ist privat und darf nicht öffentlich gemacht werden. Es enthält vertrauliche Bewerbungs- und Kontaktdaten, Referenzkontakte und private Bewerbungsunterlagen.

## Datenschutzregeln

- Produktionsdaten dürfen vollständig real sein, solange das Repository privat bleibt.
- Ein späterer Public-/Demo-Modus muss ausdrücklich anonymisierte Beispieldaten verwenden.
- Vor jeder Änderung auf öffentliche Sichtbarkeit müssen personenbezogene Daten, Referenzkontakte und private Unterlagen entfernt oder anonymisiert werden.
- Keine Passwörter, API-Schlüssel, Tokens, SSH-Schlüssel, Session-Cookies oder technischen Zugangsdaten committen.

## Qualitätsregeln

- CV-Fakten nicht erfinden.
- LinkedIn-Skills gemäss belegbarer Erfahrungsstufe klassifizieren.
- Nach Änderungen `npm run validate`, Render-Befehl und `npm run test` ausführen.
