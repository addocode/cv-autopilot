# Upload-Hinweis

Dieses Paket enthält private Bewerbungsunterlagen und Referenz-Telefonnummern.

## Vor dem Upload

1. Das Repository `addocode/cv-autopilot` auf **private** stellen.
2. Den gesamten Inhalt dieses Ordners in das Repository hochladen.
3. Danach Codex mit `CODEX_TASK.md` starten.

Falls das Repository öffentlich bleiben soll, die PDFs vorher redigieren und
private Telefonnummern/Adressen entfernen. `data/private/` und `.env` müssen
später in `.gitignore` stehen.
